# 미니 프로젝트

- ### Movieboja설명

  - 현재 서버는 분실하였고 일부 기능을 확인하시고 싶으시면 첫화면을 MainActivity로 바꾸시고 제가 빼놓은 다음 지도 api키를 입력하시면 됩니다/
  - SignActivity와 LoginActivity는 졸업작품에서 쓰였던 코드들을 재활용 한것이고 파이어베이스와 리사이클러뷰의 2개의 뷰홀더를 이용해 채팅하는 것 처럼 구현 하였습니다. retroift2를 이용해 다음에서 영화정보를 받아옵니다. 다음 지도 api를 이용해 안산 지역 영화관에 마커를 찍어 놓아고 해당 마커에 리뷰게시판을 구현 중 그만둔 프로젝트입니다. 


- ### RealmwithRetroiftwithparsingPractice 설명

  - 네이버의 단어검색을 크롤링 하여 검색한 단어를 realm을 통해 저장하는 미니프로젝트입니다. 크롤링 할때 JSOUP을 사용하지 않고 일일이 검색하여 크롤링 하였습니다.
